# Anki JS Cards - PWA для изучения JavaScript

Прогрессивное веб-приложение (PWA) для изучения JavaScript терминов в формате карточек Anki.

## 🚀 Быстрый старт

Просто откройте `index.html` в браузере или разверните на любом веб-сервере.

## 📱 Установка как PWA

### Android (Chrome):
1. Откройте сайт в Chrome
2. Нажмите «Установить» в всплывающем уведомлении
3. Или через меню → «Установить приложение»

### iOS (Safari):
1. Откройте сайт в Safari
2. Нажмите кнопку «Поделиться» (Share)
3. Выберите «На экран "Домой"» (Add to Home Screen)

### Desktop (Chrome/Edge):
1. Нажмите на иконку установки в адресной строке
2. Или через меню → «Установить приложение»

## ✨ Функции

### 🎯 Режимы изучения
- **Навигация по темам** - выбирайте конкретные темы из меню
- **Случайный режим** - перемешанные карточки из всех тем
- **Перемешивание** - случайный порядок в текущей теме

### 🔄 Карточки Anki
- **Лицевая сторона** - термин
- **Оборотная сторона** - английское название + русский перевод
- **Примеры кода** - с подсветкой синтаксиса

### 🎮 Управление
| Действие | Способ |
|----------|--------|
| Перевернуть карточку | Тап / Пробел / Enter |
| Следующая карточка | Свайп влево / → / Кнопка |
| Предыдущая карточка | Свайп вправо / ← / Кнопка |
| Отметить как изученное | Кнопка "✓ Знаю" / Клавиша 1 |
| Отметить для повторения | Кнопка "✗ Учить" / Клавиша 2 |
| Назад | ESC / Кнопка "Назад" |

### 💾 Прогресс
- Сохранение прогресса в localStorage
- Отслеживание изученных карточек (зеленые точки)
- Карточки для повторения (красные точки)

### 🎨 Темы
- Темная тема (по умолчанию)
- Светлая тема (переключение кнопкой 🌙/☀️)

## 📊 Данные

Карточки организованы по темам:
1. **Неделя 1: Основы языка** - переменные, типы данных, операторы, циклы
2. **Неделя 2: Функции и объекты** - функции, замыкания, объекты, классы
3. **Неделя 3: Массивы, строки, числа** - методы массивов, работа со строками
4. **Неделя 4: Асинхронность, DOM** - промисы, async/await, DOM API
5. **Продвинутые термины** - паттерны, TypeScript, Web APIs

Всего: **120+ карточек**

## 🛠 Технологии

- Vanilla JavaScript (ES6+)
- CSS3 с CSS Variables
- Service Workers
- LocalStorage API
- Web App Manifest

## 🔧 Локальная разработка

```bash
# Простой HTTP сервер для тестирования PWA
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000`
